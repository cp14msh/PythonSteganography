# Python Steganography Project

This project uses Python and the PIL (Pillow) library to hide secret messages
inside PNG image files.

## Features
- `encode.py`: Hides a repeating secret message in the pixels of an image.
- `decode.py`: Automatically finds and extracts the repeating secret message.

## How to Use

1.  **Encode:**
    `python encode.py`

2.  **Decode:**
    `python decode.py`
